# **App Name**: Drive Admin

## Core Features:

- Driver Onboarding: Review documents and video verification submitted via the EURO mobile application for new driver onboarding.
- Account Approval: Manage driver account approvals with necessary checks and verifications.
- Data Inquiry Tool: An AI-powered tool helps admins efficiently process and respond to driver data inquiries, automatically extracting relevant information. The tool is intended to help with reasoning.
- Feedback Management: Centralized system for collecting, reviewing, and responding to driver feedback.
- Performance Monitoring: Track driver performance metrics to identify areas for improvement.
- Driver Profiles: Comprehensive profiles for each driver, including contact details, vehicle information, and performance history.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) for a professional and trustworthy feel.
- Background color: Light gray (#F4F6F8) to provide a clean and modern backdrop.
- Accent color: Teal (#00A36C) to highlight key actions and elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body.
- Use a set of consistent and professional icons to represent different functions and data points.
- Clean and intuitive dashboard layout for easy navigation and quick access to important information.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
